Skeletons

A Brunch skeleton is basically an application boilerplate that provides a good starting point for new applications.

Creating new application with any skeleton is pretty simple:

# Simply create "dead simple" app without any opinions.
brunch new
# Example from an URL.
brunch new --skeleton https://github.com/paulmillr/brunch-with-chaplin
# There are some aliases for popular skeletons:
brunch new custom-dir -s react

Brunch doesn’t encourage you to use any particular technology, but application skeletons can start you on your way with a specific pattern.

The complete skeletons list is on GitHub. If you create your own open-source skeleton, feel free to add it!
